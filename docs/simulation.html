Interactive Simulation

Explore how the AMCL algorithm localizes a robot in real-time. Control the robot and watch particles converge as it navigates in uncertainty.

How to Interact

  • Platform: Use a desktop or laptop computer for the best experience.
  • Controls: Drive the robot using W, A, S, D or the Arrow Keys.
  • Goal: Observe the particles clustering towards the robot's true position as you move around.

Currently, parameter adjustments are unavailable. Simply control the robot and watch how AMCL manages uncertainty and converges on a stable estimate of the robot's location.
